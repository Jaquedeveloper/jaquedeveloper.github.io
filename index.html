NETWORKS APP

Django App for network graph creation based on csv files upload. The software allows the registration of users. The users can upload csv files and it will generate the network graph based on the nodes and vertices of the file and calculate the network metrics (centrality, betweeness). The user can interact with the graph and download the csv file with the metrics and the pictures. There are two different types of visualization and the user can switch them while interacing with the network graph.
